# Naming Convention Enforcer

Checks if code elements (variables, functions, classes) adhere to established naming conventions.

## Free
```yaml
- uses: walshd1/naming-convention-enforcer@v1
  with:
    gemini_api_key: ${{ secrets.GEMINI_API_KEY }}
```

## Paid (cost + 4.75%)
```yaml
- uses: walshd1/naming-convention-enforcer@v1
  with:
    service_token: ${{ secrets.ACTION_FACTORY_TOKEN }}
```
